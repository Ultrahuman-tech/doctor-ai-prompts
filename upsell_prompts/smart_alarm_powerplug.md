 Select when the assistant's response discusses wake timing, alarm optimization, or sleep cycles.

Trigger when:
  - User asks about optimal wake-up time
  - User mentions feeling groggy or tired when waking up
  - User asks about sleep cycles and when to wake
  - User discusses difficulty waking up refreshed
  - User asks about alarms that work with sleep stages
  - Assistant recommends optimizing wake timing based on sleep data

Do NOT trigger when:
  - User is asking about general sleep quality without wake concerns
  - Conversation is about bedtime rather than wake time
